## What is React

React is a JavaScript library for building user interfaces, originally
developed at Facebook and released as open source in 2013. It introduced a
component-based model in which the interface is described as a tree of small,
reusable pieces, each responsible for rendering one part of the screen.

At the heart of React is the virtual DOM, an in-memory representation of the
real document. When state changes, React computes the minimal set of updates
required and applies only those, which keeps interfaces fast even as they
grow complicated. This diffing approach was a significant departure from the
imperative DOM manipulation styles that came before it.

React components can be written as functions or classes, though modern React
strongly favors functions combined with hooks. Hooks such as useState and
useEffect let function components hold local state and perform side effects,
which previously required class components with lifecycle methods.

The ecosystem around React is vast: routing, state management, data fetching,
and styling all have multiple mature solutions. Understanding the core ideas
of components, props, state, and one-way data flow is the foundation on which
all of that tooling rests.
